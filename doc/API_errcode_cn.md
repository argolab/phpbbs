附录A  错误码对照表
-----------------------------

     目前模块码如下：
     - 01 System error
     - 02 Section
     - 03 User
     - 04 Board
     - 05 Post
     - 06 Mail
     - 07 Misc
     - 08 Admin
     - 09 Message
